# v1.0.10

Released: 2026-06-11

## Fixed

- Changed `PermissionCache.invalidateAll()` to delete only `permission-core:rules:*` keys instead of clearing the whole shared cache backend.
- Added owned-cache cleanup on `PermissionCore.close()` so internally created `cache-hub` memory caches are destroyed during runtime shutdown.
- Fixed light-mode documentation code blocks so they use a light payment-console surface while keeping dark mode code blocks dark.

## Changed

- Reorganized the Rspress documentation site into English default pages and `/zh/` Chinese pages.
- Translated the root README to English and added the published documentation link.
- Added a fintech/payment-oriented documentation theme and replaced the hero bitmap with code-native SVG authorization artwork.
- Upgraded the verified production dependency line to `cache-hub@2.2.4 + monsqlize@2.0.3`.
- Updated the Vitest toolchain to `3.2.6`.
- Added package `homepage` / `bugs` metadata.
- Added `test:audit`, `test:all`, and `test:all:with-audit`, and made `prepublishOnly` run the full release gate.
